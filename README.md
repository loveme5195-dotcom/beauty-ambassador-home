# MÉLISSA Luxury Ambassador Portfolio

以 12 張精品風格形象照製作的單頁個人主頁，使用 HTML、CSS、ES6 與 Bootstrap 5。

## 預覽

直接開啟 `index.html`，或在此目錄啟動本地靜態伺服器。

## 自訂

- 姓名與文案：編輯 `index.html`
- 色彩與排版：編輯 `styles.css` 頂部變數
- 篩選與燈箱：編輯 `app.js`
- 聯絡資料：將 `hello@example.com` 與社群連結替換成正式資訊

## 圖片

網站以相對路徑引用相鄰的 `chanel-ambassador-6` 與 `hermes-ambassador-6` 圖片資料夾，請勿單獨移動網站資料夾。
